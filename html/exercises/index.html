Watching anime has been a favorite pass time of mine for almost a decade now. Sometimes there are many shows
that I'd like to watch at the same time, some that release new episodes on a weekly basis and others that have new seasons
that are already completed. This can make watching all of my shows a bit of a daunting task.

To resolve this problem, I'd like to create a website which organizes all of the shows that I am interested using different categories.
For example, some categories would be:

  • Genre
  • Streaming Service
  • Currently airing or already complete
  • Number of episodes watched/remaining
